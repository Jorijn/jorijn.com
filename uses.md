⚡ People working in the IT are only as powerful as the tools they use for their job. I love to learn from others and improve my own workflow. This page serves as a living document of the things I use every day.

## Desktop
 * [PHPStorm](https://www.jetbrains.com/phpstorm/): The number 1 development IDE I use every day. It makes me a better developer. I use [Material Theme UI](https://plugins.jetbrains.com/plugin/8006-material-theme-ui) and [Dank Mono](https://dank.sh/). There's also a [screenshot](/content/images/2018/07/phpstorm-screenshot.png) of my setup.
 * [Chrome](https://www.google.nl/chrome/): Favorite web browser, integrates nicely as I'm mostly hosted on Google architecture.
 * [Things](https://culturedcode.com/things/): Great TODO application for Mac and iOS. It's the only GTD software I'm able to stick with.
 * [Telegram](https://telegram.org/): In many ways a better chat application than WhatsApp.
 * [iTerm 2](https://www.iterm2.com/): Terminal replacement app for Mac OS X. Also using [Dank Mono](https://dank.sh/) with the [Tomorrow Night](https://github.com/chriskempson/tomorrow-theme) theme. [Screenshot](/content/images/2018/07/iterm-screenshot.png).
     * [Oh-My-ZSH](https://github.com/robbyrussell/oh-my-zsh): Framework for making your ZSH shell awesome.
     * [Pure Prompt](https://github.com/sindresorhus/pure): Pretty, minimal and fast ZSH prompt
     * [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions): Fish-like autosuggestions for ZSH.
     * [zsh-syntax-highlighting](https://github.com/zsh-users/zsh-syntax-highlighting): Fish shell like syntax highlighting for ZSH.
     * [autojump](https://github.com/wting/autojump): A cd command that learns - easily navigate directories from the command line
 * [TweetBot](https://tapbots.com/tweetbot/mac/): The best Twitter client for Mac OS X.
 * [Spotify](https://www.spotify.com/): If I'm awake, there's playing music. 
 * [Amphetamine](https://itunes.apple.com/nl/app/amphetamine/id937984704?mt=12): Little tool for Mac OS X to keep your mac awake.
 * [Dropshare](https://getdropsha.re/): Tool for taking screenshots and automatically hosting them on a platform of your choice.
 * [Viscosity](https://www.sparklabs.com/viscosity/): Nice OpenVPN client for Mac OS X.
 * [1Password](https://1password.com/): I have over thousand passwords in 1Password and it's still fast, good looking and going strong. 
 * [Unsplash Wallpapers](https://apps.apple.com/us/app/unsplash-wallpapers/id1284863847?mt=12): Nice wallpapers, refreshed every day  

## Hardware
 * [Apple MacBook Pro](https://www.apple.com/nl/macbook-pro/specs-2015/): 15-inch, 2015, 2,8 GHz i7, 16 GB RAM. I seriously dislike where Apple is going with their USB-C and Touchbar stuff. Recently bought this 2015 generation and could not be happier. 
 * [Apple iPad Pro](https://www.apple.com/ipad-pro/): 11-inch display, Apple Pencil and Apple Keyboard Case. This is great for gaming, watching movies and taking notes during meetings. 
 * [Apple AirPods](https://www.apple.com/nl/shop/product/MMEF2ZM/A/airpods): I use these awesome little devices for calling hands free without the messy wires. I also use them while running.
 * [Apple iPhone X](https://www.apple.com/nl/iphone-x/): Coming from an iPhone 7 I took the plunge for this awesome phone. I'm still hesitant about whether I like FaceID but the huge amount of screen and lack of buttons is really great.
 * [Apple Watch Series 4](https://www.apple.com/apple-watch-series-4/): Before this I owned a Fitbit Charge HR 2, this watch reminds me about being active, provides useful information at all times and looks a lot better too.
 * [Apple Magic Keyboard](https://www.apple.com/nl/shop/product/MLA22N/A/magic-keyboard-nederlands): Tiny keyboard that types amazingly. Big plus is that it fits in my bag to go.
 * [Logitech MX Master 2S](https://www.logitech.com/nl-nl/product/mx-master-2s-flow): A big mouse with multiple useful buttons. Ergonomically designed.
 * [Yubikey 4](https://www.yubico.com/product/yubikey-4-series/): USB-A keychain device for U2F security. I'm also using this with OpenPGP as smartcard to safely store my SSH keys.
 * [Roost Laptop Stand](https://www.therooststand.com/): One of the worst things you can do for your posture is working on a laptop. This laptop stand elevates my laptop to a proper height. It's lightweight and also fits in my bag.

## Services
 * [Docker](https://www.docker.com/): I create specific containerized infrastructure per-client using Docker(-compose).
 * [Gitea](https://gitea.io/en-us/): Nice and lightweight Git service hosted on my Synology NAS using Docker.
 * [Miniflux](https://miniflux.app/): RSS reader, self-hosted on my Synology NAS using Docker.
 * [Moneybird](https://www.moneybird.com/): Invoicing software for entrepreneurs. Dutch made!
